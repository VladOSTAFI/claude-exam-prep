## Task 5.1: Manage Context Windows Effectively

### Understanding Token Limits and Model Capabilities

Claude models have different context window sizes:

| Model | Context Window | Input Pricing | Output Pricing |
|-------|---|---|---|
| Claude 3.5 Haiku | 200K tokens | $0.80/1M | $4.00/1M |
| Claude 3.5 Sonnet | 200K tokens | $3.00/1M | $15.00/1M |
| Claude 3.5 Opus | 200K tokens | $15.00/1M | $75.00/1M |

Key considerations:
- **Input tokens**: Count toward context limit and cost
- **Output tokens**: Generated by the model, cost more than input
- **Cache usage**: First 5 minutes of cached content = 90% discount; repeated access = 10% of input cost
- **Planning**: Always leave buffer tokens for output (typically 10-20% of available context)

### Strategy 1: Conversation Summarization

For long-running conversations, periodically summarize context to maintain coherence while freeing tokens.

## Task 5.2: Design Reliable Error Handling and Escalation

### Error Classification Framework

Different errors require different responses:

```typescript
enum ErrorSeverity {
  RECOVERABLE = 'recoverable',      // Retry can help
  DEGRADED = 'degraded',            // Partial success possible
  CRITICAL = 'critical',            // Must escalate
  FATAL = 'fatal'                   // System cannot continue
}

interface AgenError {
  type: string;
  severity: ErrorSeverity;
  message: string;
  context: Record<string, unknown>;
  timestamp: Date;
  retryable: boolean;
  escalationThreshold: number;
}

class ErrorClassifier {
  classify(error: Error): AgenError {
    if (error instanceof RateLimitError) {
      return {
        type: 'RATE_LIMIT',
        severity: ErrorSeverity.RECOVERABLE,
        message: error.message,
        context: { remainingQuota: error.remainingQuota },
        timestamp: new Date(),
        retryable: true,
        escalationThreshold: 3
      };
    }

    if (error instanceof ValidationError) {
      return {
        type: 'VALIDATION',
        severity: ErrorSeverity.DEGRADED,
        message: error.message,
        context: { invalidFields: error.fields },
        timestamp: new Date(),
        retryable: false,
        escalationThreshold: 1
      };
    }

    if (error instanceof TimeoutError) {
      return {
        type: 'TIMEOUT',
        severity: ErrorSeverity.RECOVERABLE,
        message: error.message,
        context: { timeoutMs: error.timeoutMs },
        timestamp: new Date(),
        retryable: true,
        escalationThreshold: 5
      };
    }

    return {
      type: 'UNKNOWN',
      severity: ErrorSeverity.CRITICAL,
      message: error.message,
      context: {},
      timestamp: new Date(),
      retryable: false,
      escalationThreshold: 1
    };
  }
}

class RateLimitError extends Error {
  constructor(public remainingQuota: number) {
    super('Rate limit exceeded');
  }
}

class ValidationError extends Error {
  constructor(public fields: string[]) {
    super('Validation failed');
  }
}

class TimeoutError extends Error {
  constructor(public timeoutMs: number) {
    super('Operation timed out');
  }
}
```

### Escalation Threshold System

```typescript
interface EscalationQueue {
  taskId: string;
  error: AgenError;
  retryCount: number;
  humanReviewRequired: boolean;
  reviewReason: string;
  escalatedAt: Date;
  priority: 'low' | 'medium' | 'high' | 'critical';
}

class EscalationManager {
  private escalationQueue: EscalationQueue[] = [];
  private errorClassifier = new ErrorClassifier();

  async handleError(
    taskId: string,
    error: Error,
    retryCount: number
  ): Promise<'retry' | 'escalate' | 'fail'> {
    const classifiedError = this.errorClassifier.classify(error);

    // Check if we should escalate
    if (retryCount >= classifiedError.escalationThreshold) {
      await this.escalate(taskId, classifiedError);
      return 'escalate';
    }

    // Check if retryable
    if (!classifiedError.retryable) {
      await this.escalate(taskId, classifiedError, 'Non-retryable error');
      return 'escalate';
    }

    // Otherwise, allow retry
    return 'retry';
  }

  private async escalate(
    taskId: string,
    error: AgenError,
    reason?: string
  ): Promise<void> {
    const escalation: EscalationQueue = {
      taskId,
      error,
      retryCount: 0,
      humanReviewRequired: error.severity === ErrorSeverity.CRITICAL,
      reviewReason: reason || `Error severity: ${error.severity}`,
      escalatedAt: new Date(),
      priority: this.determinePriority(error.severity)
    };

    this.escalationQueue.push(escalation);
    await this.notifyOperations(escalation);
  }

  private determinePriority(severity: ErrorSeverity): 'low' | 'medium' | 'high' | 'critical' {
    switch (severity) {
      case ErrorSeverity.RECOVERABLE: return 'low';
      case ErrorSeverity.DEGRADED: return 'medium';
      case ErrorSeverity.CRITICAL: return 'high';
      case ErrorSeverity.FATAL: return 'critical';
    }
  }

  private async notifyOperations(escalation: EscalationQueue): Promise<void> {
    // Send to monitoring system, create ticket, etc.
    console.log(`[ESCALATION] Task ${escalation.taskId}: ${escalation.reviewReason}`);
  }

  getEscalationQueue(): EscalationQueue[] {
    return this.escalationQueue;
  }
}
```

### Structured Handoff Protocol

```typescript
interface AgentHandoff {
  fromAgent: string;
  toAgent: string;
  handoffId: string;
  timestamp: Date;
  originalRequest: string;
  executionContext: Record<string, unknown>;
  completedWork: string;
  nextSteps: string;
  tokenCountUsed: number;
  tokensRemaining: number;
  confidenceScore: number;
  metadata: HandoffMetadata;
}

interface HandoffMetadata {
  priority: 'low' | 'medium' | 'high';
  escalation: boolean;
  requiresHumanReview: boolean;
  partialResults: boolean;
  errorLog: string[];
}

class HandoffManager {
  private handoffHistory: AgentHandoff[] = [];

  async createHandoff(
    fromAgent: string,
    toAgent: string,
    context: Record<string, unknown>,
    completedWork: string,
    nextSteps: string,
    tokenMetrics: { used: number; remaining: number }
  ): Promise<AgentHandoff> {
    const handoffId = this.generateHandoffId();

    // Compress context for downstream agent
    const compressedContext = await this.compressContextForHandoff(
      context,
      tokenMetrics.remaining
    );

    const handoff: AgentHandoff = {
      fromAgent,
      toAgent,
      handoffId,
      timestamp: new Date(),
      originalRequest: context.originalRequest as string,
      executionContext: compressedContext,
      completedWork,
      nextSteps,
      tokenCountUsed: tokenMetrics.used,
      tokensRemaining: tokenMetrics.remaining,
      confidenceScore: this.calculateConfidence(completedWork),
      metadata: {
        priority: this.determinePriority(context),
        escalation: false,
        requiresHumanReview: false,
        partialResults: false,
        errorLog: []
      }
    };

    this.handoffHistory.push(handoff);
    return handoff;
  }

  private async compressContextForHandoff(
    context: Record<string, unknown>,
    tokensAvailable: number
  ): Promise<Record<string, unknown>> {
    // Estimate if we need to compress
    const contextString = JSON.stringify(context);
    const estimatedTokens = Math.ceil(contextString.length / 4);

    if (estimatedTokens > tokensAvailable * 0.3) {
      // Too large, compress by removing non-essential keys
      const compressed: Record<string, unknown> = {};
      const essentialKeys = [
        'originalRequest',
        'decisions',
        'constraints',
        'dependencies'
      ];

      for (const key of essentialKeys) {
        if (key in context) {
          compressed[key] = context[key];
        }
      }

      return compressed;
    }

    return context;
  }

  private calculateConfidence(completedWork: string): number {
    // Simple heuristic: longer, more detailed work = higher confidence
    const workLength = completedWork.length;
    if (workLength > 2000) return 0.9;
    if (workLength > 1000) return 0.75;
    if (workLength > 500) return 0.6;
    return 0.4;
  }

  private determinePriority(context: Record<string, unknown>): 'low' | 'medium' | 'high' {
    const priority = context.priority as string;
    if (priority === 'critical') return 'high';
    if (priority === 'normal') return 'medium';
    return 'low';
  }

  private generateHandoffId(): string {
    return `ho_${Date.now()}_${Math.random().toString(36).substr(2, 9)}`;
  }

  async receiveHandoff(handoffId: string): Promise<AgentHandoff | null> {
    return this.handoffHistory.find(h => h.handoffId === handoffId) || null;
  }

  getHandoffAuditTrail(handoffId: string): AgentHandoff[] {
    // Return all handoffs related to a specific task
    return this.handoffHistory.filter(h =>
      h.handoffId === handoffId || h.metadata.errorLog.includes(handoffId)
    );
  }
}
```

### Multi-Agent Coordinator

```typescript
interface Agent {
  name: string;
  capabilities: string[];
  maxContextTokens: number;
  isAvailable: boolean;
}

interface Task {
  id: string;
  description: string;
  requiredCapabilities: string[];
  currentAgent: string | null;
  completedStages: string[];
  status: 'pending' | 'in_progress' | 'awaiting_handoff' | 'completed' | 'failed';
}

class MultiAgentCoordinator {
  private agents: Map<string, Agent> = new Map();
  private tasks: Map<string, Task> = new Map();
  private handoffManager: HandoffManager;

  constructor(handoffManager: HandoffManager) {
    this.handoffManager = handoffManager;
  }

  registerAgent(agent: Agent): void {
    this.agents.set(agent.name, agent);
  }

  async routeTask(task: Task, context: Record<string, unknown>): Promise<void> {
    // Find best agent for task
    const bestAgent = this.findBestAgent(task);

    if (!bestAgent) {
      task.status = 'failed';
      return;
    }

    task.currentAgent = bestAgent.name;
    task.status = 'in_progress';

    // Execute task with agent
    const result = await this.executeWithAgent(bestAgent, task, context);

    // Check if handoff needed
    if (result.requiresHandoff) {
      const nextAgent = this.findNextAgent(task, bestAgent);
      if (nextAgent) {
        await this.performHandoff(task, bestAgent, nextAgent, result.context, context);
      } else {
        task.status = 'completed';
      }
    } else {
      task.status = 'completed';
    }
  }

  private findBestAgent(task: Task): Agent | null {
    for (const capability of task.requiredCapabilities) {
      for (const agent of this.agents.values()) {
        if (
          agent.capabilities.includes(capability) &&
          agent.isAvailable
        ) {
          return agent;
        }
      }
    }
    return null;
  }

  private findNextAgent(task: Task, currentAgent: Agent): Agent | null {
    // Find agent for next required capability
    const remainingCapabilities = task.requiredCapabilities.filter(
      c => !task.completedStages.includes(c)
    );

    if (remainingCapabilities.length === 0) return null;

    for (const capability of remainingCapabilities) {
      for (const agent of this.agents.values()) {
        if (
          agent.name !== currentAgent.name &&
          agent.capabilities.includes(capability) &&
          agent.isAvailable
        ) {
          return agent;
        }
      }
    }
    return null;
  }

  private async executeWithAgent(
    agent: Agent,
    task: Task,
    context: Record<string, unknown>
  ): Promise<{ requiresHandoff: boolean; context: Record<string, unknown> }> {
    // Call agent with task and context
    // Return whether handoff is needed
    return { requiresHandoff: false, context };
  }

  private async performHandoff(
    task: Task,
    fromAgent: Agent,
    toAgent: Agent,
    executionContext: Record<string, unknown>,
    originalContext: Record<string, unknown>
  ): Promise<void> {
    const handoff = await this.handoffManager.createHandoff(
      fromAgent.name,
      toAgent.name,
      executionContext,
      `Completed stages: ${task.completedStages.join(', ')}`,
      `Next: Handle requirements for ${toAgent.name}`,
      { used: 50000, remaining: 150000 }
    );

    task.currentAgent = toAgent.name;
    task.status = 'awaiting_handoff';

    // Execute with next agent
    const nextResult = await this.executeWithAgent(toAgent, task, executionContext);

    if (!nextResult.requiresHandoff) {
      task.status = 'completed';
    } else {
      // Continue routing
```

```typescript
interface ModelPricing {

const modelPrices: Record<string, ModelPricing> = {
  'claude-3-5-haiku': {

    outputPrice: 4.00,

    cacheReadPrice: 0.15

class TokenCostCalculator {
  calculateCost(
    model: string,
    inputTokens: number,
    outputTokens: number,
    cacheReadTokens: number = 0,
    cacheWriteTokens: number = 0
  ): number {
    const pricing = modelPrices[model];
    if (!pricing) throw new Error(`Unknown model: ${model}`);

    const inputCost = (inputTokens / 1000000) * pricing.inputPrice;
    const outputCost = (outputTokens / 1000000) * pricing.outputPrice;
    const cacheReadCost = (cacheReadTokens / 1000000) * pricing.cacheReadPrice;
    const cacheWriteCost = (cacheWriteTokens / 1000000) * pricing.cacheInputPrice;

    return inputCost + outputCost + cacheReadCost + cacheWriteCost;
  }

  selectOptimalModel(
    inputTokens: number,
    estimatedOutputTokens: number,
    performanceRequired: 'low' | 'medium' | 'high',
    costSensitivity: 'high' | 'medium' | 'low'
  ): string {
    // If cost-sensitive and low performance needed, use Haiku
    if (costSensitivity === 'high' && performanceRequired === 'low') {
      return 'claude-3-5-haiku';
    }

    // If high performance needed, use Opus
    if (performanceRequired === 'high') {
      return 'claude-3-5-opus';
    }

    // Default to Sonnet for balanced performance/cost
    return 'claude-3-5-sonnet';
  }

  estimateQuarterlyCost(
    requestsPerDay: number,
    avgInputTokens: number,
    avgOutputTokens: number,
    model: string = 'claude-3-5-sonnet'
  ): number {
    const dailyCost = this.calculateCost(
      model,
      avgInputTokens,
      avgOutputTokens
    );
    const annualCost = dailyCost * requestsPerDay * 365;
    return annualCost / 4;
```

### Prompt Caching Strategy

```typescript
interface CachedPrompt {
  id: string;
  content: string;
  hash: string;
  createdAt: Date;
  hitCount: number;
  lastAccessed: Date;
  tokens: number;
}

class PromptCacheManager {
  private cache: Map<string, CachedPrompt> = new Map();
  private costCalculator = new TokenCostCalculator();

  hashPrompt(content: string): string {
    // In practice, use cryptographic hash
    return Buffer.from(content).toString('base64').slice(0, 16);
  }

  addToCache(content: string, estimatedTokens: number): string {
    const hash = this.hashPrompt(content);
    const existing = Array.from(this.cache.values()).find(c => c.hash === hash);

    if (existing) {
      existing.hitCount++;
      existing.lastAccessed = new Date();
      return existing.id;
    }

    const id = `cached_${Date.now()}`;
    this.cache.set(id, {
      id,
      content,
      hash,
      createdAt: new Date(),
      hitCount: 0,
      lastAccessed: new Date(),
      tokens: estimatedTokens
    });

    return id;
  }

  getCacheUtilization(): {
    totalCachedPrompts: number;
    totalTokensCached: number;
    estimatedMonthlySavings: number;
  } {
    const totalCachedPrompts = this.cache.size;
    const totalTokensCached = Array.from(this.cache.values()).reduce(
      (sum, p) => sum + p.tokens,
      0
    );

    // Cache read hits save 90% on input tokens
    const totalHits = Array.from(this.cache.values()).reduce(
      (sum, p) => sum + p.hitCount,
      0
    );
    const savedTokens = totalHits * 100; // Average tokens per hit

    const savingsWithoutCache = this.costCalculator.calculateCost(
      'claude-3-5-sonnet',
      savedTokens,
      0
    );
    const savingsWithCache = this.costCalculator.calculateCost(
      'claude-3-5-sonnet',
      savedTokens,
      0,
      savedTokens // These become cache reads
    );

    return {
      totalCachedPrompts,
      totalTokensCached,
      estimatedMonthlySavings: (savingsWithoutCache - savingsWithCache) * 30
    };
  }

  // Example: Cache system prompts that are reused
  cacheSystemPrompt(role: string, description: string): string {
    const systemPrompt = `You are a ${role}. ${description}`;
    return this.addToCache(systemPrompt, Math.ceil(systemPrompt.length / 4));
  }
}
```

### Dynamic Model Selection

```typescript
interface RequestProfile {
  complexity: 'low' | 'medium' | 'high';
  latencyRequirement: 'instant' | 'normal' | 'relaxed';
  tokenEstimate: number;
  budgetPerRequest: number | null;
}

class DynamicModelSelector {
  private calculator = new TokenCostCalculator();

  selectModel(profile: RequestProfile): string {
    // Algorithm: Consider complexity, latency, tokens, and budget

    // If budget is very tight, prefer Haiku
    if (profile.budgetPerRequest && profile.budgetPerRequest < 0.01) {
      return 'claude-3-5-haiku';
    }

    // If instant latency needed, prefer Haiku (faster)
    if (profile.latencyRequirement === 'instant') {
      return 'claude-3-5-haiku';
    }

    // If high complexity, need Opus
    if (profile.complexity === 'high') {
      return 'claude-3-5-opus';
    }

    // If high tokens and relaxed latency, could use Haiku to save costs
    if (
      profile.latencyRequirement === 'relaxed' &&
      profile.tokenEstimate > 100000
    ) {
      return 'claude-3-5-haiku';
    }

    // Default: balanced choice
    return 'claude-3-5-sonnet';
  }

  shouldUseCache(
    promptTokens: number,
    expectedReuses: number
  ): boolean {
    // Cache if prompt is large and will be reused
    if (promptTokens < 1024) return false; // Too small to cache

    // Calculate break-even
    const cacheWriteCost = this.calculator.calculateCost(
      'claude-3-5-sonnet',
      promptTokens,
      0,
      0,
      promptTokens
    );
    const directCost = this.calculator.calculateCost(
      'claude-3-5-sonnet',
      promptTokens,
      0
    );
    const cacheCost = cacheWriteCost + (this.calculator.calculateCost(
      'claude-3-5-sonnet',
      promptTokens,
      0,
      promptTokens // Cache read
    ) * expectedReuses);

    return cacheCost < (directCost * (expectedReuses + 1));
  }
}
```

```typescript
interface Budget {
  month: string;
  limit: number;

  remaining: number;

class CostMonitor {
  private transactions: Array<{
    timestamp: Date;
    model: string;
    cost: number;
    inputTokens: number;
    outputTokens: number;
  }> = [];
  private monthlyBudget: number = 1000;

  recordTransaction(
    model: string,
    inputTokens: number,
    outputTokens: number,
    calculator: TokenCostCalculator
  ): void {
    const cost = calculator.calculateCost(model, inputTokens, outputTokens);

    this.transactions.push({
      timestamp: new Date(),
      model,
      cost,
      inputTokens,
      outputTokens
    });

    // Alert if over budget
    if (this.getCurrentMonthSpent() > this.monthlyBudget) {
      this.triggerBudgetAlert();
    }
  }

  getCurrentMonthBudget(): Budget {
    const now = new Date();
    const monthKey = `${now.getFullYear()}-${String(now.getMonth() + 1).padStart(2, '0')}`;

    const monthlyTransactions = this.transactions.filter(t => {
      const tMonth = `${t.timestamp.getFullYear()}-${String(t.timestamp.getMonth() + 1).padStart(2, '0')}`;
      return tMonth === monthKey;
    });

    const spent = monthlyTransactions.reduce((sum, t) => sum + t.cost, 0);

    return {
      month: monthKey,
      limit: this.monthlyBudget,
      spent,
      remaining: this.monthlyBudget - spent,
      percentUsed: (spent / this.monthlyBudget) * 100
    };
  }

  private getCurrentMonthSpent(): number {
    return this.getCurrentMonthBudget().spent;
  }

  private triggerBudgetAlert(): void {
    const budget = this.getCurrentMonthBudget();
    console.warn(
      `Budget alert: ${budget.percentUsed.toFixed(1)}% of monthly budget used (${budget.spent.toFixed(2)} / ${budget.limit})`
    );
  }

  getCostBreakdown(): Record<string, number> {
    const breakdown: Record<string, number> = {};

    this.transactions.forEach(t => {
      breakdown[t.model] = (breakdown[t.model] || 0) + t.cost;
    });

    return breakdown;
  }
}
```

### Retry Strategy with Exponential Backoff

```typescript
interface RetryPolicy {
  maxRetries: number;
  initialDelayMs: number;
  maxDelayMs: number;
  backoffMultiplier: number;
  jitterFactor: number; // 0-1, adds randomness to prevent thundering herd
}

class RetryHandler {
  private defaultPolicy: RetryPolicy = {
    maxRetries: 3,
    initialDelayMs: 100,
    maxDelayMs: 30000,
    backoffMultiplier: 2,
    jitterFactor: 0.1
  };

  async executeWithRetry<T>(
    operation: () => Promise<T>,
    policy: Partial<RetryPolicy> = {}
  ): Promise<T> {
    const mergedPolicy = { ...this.defaultPolicy, ...policy };
    let lastError: Error | null = null;

    for (let attempt = 0; attempt <= mergedPolicy.maxRetries; attempt++) {
      try {
        return await operation();
      } catch (error) {
        lastError = error as Error;

        if (attempt === mergedPolicy.maxRetries) {
          throw error;
        }

        const delayMs = this.calculateBackoffDelay(
          attempt,
          mergedPolicy
        );

        console.log(
          `Retry attempt ${attempt + 1}/${mergedPolicy.maxRetries} after ${delayMs}ms: ${lastError.message}`
        );

        await this.delay(delayMs);
      }
    }

    throw lastError;
  }

  private calculateBackoffDelay(attempt: number, policy: RetryPolicy): number {
    const exponentialDelay = policy.initialDelayMs * Math.pow(policy.backoffMultiplier, attempt);
    const cappedDelay = Math.min(exponentialDelay, policy.maxDelayMs);

    // Add jitter
    const jitter = cappedDelay * policy.jitterFactor * Math.random();
    return Math.floor(cappedDelay + jitter);
  }

  private delay(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}
```

### Rate Limiting and Quota Management

```typescript
interface RateLimitConfig {
  requestsPerSecond: number;
  requestsPerMinute: number;
  requestsPerHour: number;
  concurrentRequests: number;
}

class RateLimiter {
  private requestTimestamps: number[] = [];
  private concurrentCount: number = 0;
  private config: RateLimitConfig;

  constructor(config: RateLimitConfig) {
    this.config = config;
  }

  async acquireToken(): Promise<void> {
    // Check concurrent limit
    while (this.concurrentCount >= this.config.concurrentRequests) {
      await this.delay(100);
    }

    // Check rate limits
    const now = Date.now();
    this.requestTimestamps = this.requestTimestamps.filter(t => t > now - 3600000); // 1 hour window

    const recentSecond = this.requestTimestamps.filter(t => t > now - 1000).length;
    const recentMinute = this.requestTimestamps.filter(t => t > now - 60000).length;
    const recentHour = this.requestTimestamps.length;

    if (recentSecond >= this.config.requestsPerSecond) {
      await this.delay(100);
      return this.acquireToken();
    }

    if (recentMinute >= this.config.requestsPerMinute) {
      await this.delay(500);
      return this.acquireToken();
    }

    if (recentHour >= this.config.requestsPerHour) {
      const resetTime = Math.ceil((this.requestTimestamps[0] + 3600000 - now) / 1000);
      throw new Error(`Rate limit exceeded. Reset in ${resetTime}s`);
    }

    this.requestTimestamps.push(now);
    this.concurrentCount++;
  }

  releaseToken(): void {
    this.concurrentCount--;
  }

  private delay(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }

  getStats(): {
    totalRequests: number;
    concurrentRequests: number;
    utilizationPercent: number;
  } {
    return {
      totalRequests: this.requestTimestamps.length,
      concurrentRequests: this.concurrentCount,
      utilizationPercent: (this.concurrentCount / this.config.concurrentRequests) * 100
    };
```

## Key Exam Tips

1. **Context Window Arithmetic**: Always calculate token budgets. Reserve 10-20% for output. Example: With 200K tokens, keep input to ~160K to leave 40K for output.

2. **Summarization Timing**: Summarize before hitting 80% of context limit, not 100%. This gives time to process and complete current operation.

3. **Error Escalation**: Route based on severity, not just retry count. A validation error on first try should escalate faster than a transient timeout.

4. **Handoff Protocol**: Always pass structured context. Include:
   - Original request (why we're doing this)
   - Decisions made so far (what was tried)
   - Current status (what's complete)
   - Next steps (what's needed)

5. **Cost Optimization**: Cache is best for repeated system prompts and few-shot examples (100+ reuses). Not worth it for one-time prompts.

6. **Model Selection**: Haiku for simple tasks (classification, parsing). Sonnet for complex reasoning. Opus only when Sonnet fails.

7. **Logging for Observability**: Include trace IDs across handoffs. Structure logs as JSON for parsing. Log before/after state for debugging.

8. **Retry Strategy**: Use exponential backoff with jitter. Max retries varies by error type (rate limit: 5, validation: 1, timeout: 3).

9. **Rate Limiting**: Implement client-side rate limiting even with API quotas. Prevents thundering herd on service degradation.

10. **Testing Agents**: Create test harnesses for each agent capability. Include happy path, error cases, and edge cases. Validate output structure.
